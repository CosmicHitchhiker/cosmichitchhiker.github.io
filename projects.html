los_to_rc

A tool to calculate the rotation curve of galaxy using measured line of sight velocities. It can work with different types of measurements (long slit spectroscopy, IFU, or other).

find_slit

A tool to clarify the position of a slit in the long-slit spectral observations of an extended object. One needs a photometry of the object as a fits file with correct WCS to do so.

std_telluric_A0V

This catalogue of A0V stars is being used as telluric standards at the Caucasus Mountain Observatory.

andor-cameras

This is a tool to control andor cameras via TCP/IP protocol on linux systems.
